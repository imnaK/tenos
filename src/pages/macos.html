OS X / Mac OS

macOS ist das Betriebssystem, das von Apple für seine Computer entwickelt wurde. Es zeichnet sich durch eine nahtlose Integration von Hardware und Software aus. Die Benutzeroberfläche ist intuitiv gestaltet, und das System bietet fortschrittliche Sicherheitsfunktionen. Eine besondere Stärke liegt in der reibungslosen Zusammenarbeit mit anderen Apple-Geräten, was die Nutzung im Apple-Ökosystem erleichtert. macOS ist für verschiedene Anwendungsbereiche geeignet, von kreativen Projekten bis zur alltäglichen Produktivität. Es bietet eine stabile und benutzerfreundliche Plattform für Nutzer, die Wert auf eine harmonische Verbindung von Technologie und Design legen.

Mac OS 7.0
23.03.1995

macOS 7 brachte bedeutende Verbesserungen für Mac-Computer. Die Einführung von Farbunterstützung und TrueType-Schriftarten verbesserte die visuelle Erfahrung erheblich. Multithreading ermöglichte das gleichzeitige Ausführen mehrerer Anwendungen, und das Macintosh Performa 600 führte als erstes Mac-Modell ein CD-ROM-Laufwerk ein.

Mac OS 8.0
26.07.1997

macOS 8 war ein Meilenstein. Verbesserte Leistung, erweiterte Unterstützung für Netzwerkprotokolle und die Einführung von Sherlock, einer Desktop-Suchmaschine, stärkten die Funktionalität. Die Integration von USB und die Möglichkeit, den Papierkorb sofort zu leeren, waren praktische Neuerungen.

Mac OS 9.0
13.10.1999

macOS 9 führte den „Platinum“-Look ein und verbesserte die Stabilität. Die Integration von Keychain zur sicheren Verwaltung von Passwörtern und die Einführung von iTools legten den Grundstein für zukünftige Entwicklungen.

Mac OS X 10.0 (Cheetah)
24.03.2001

Cheetah, das 2001 debütierte, markiert einen Meilenstein in der Geschichte von macOS. Es markierte den Übergang von der klassischen Mac OS-Linie zu einem neuen, auf Unix basierenden Betriebssystem. Cheetah war geprägt von einer grundlegenden Umgestaltung der Benutzeroberfläche und führte die Anwender in eine Ära ein, die von einer verbesserten Stabilität und Leistung geprägt war. Die Einführung von Features wie dem neuen Dock und einer verbesserten Dateiverwaltung legte den Grundstein für die weiteren Entwicklungen von macOS.

Mac OS X 10.1 (Puma)
25.09.2001

Mit der Einführung von macOS 10.1 setzte Apple den Fokus auf die Verbesserung der Leistung und das Hinzufügen zusätzlicher Funktionen. Eine bemerkenswerte Ergänzung war die Einführung der Finder-Sidebar, die die Navigation im Dateisystem erheblich erleichterte. Darüber hinaus ermöglichte Puma das direkte Brennen von CDs vom Desktop aus, was die Handhabung von Medien optimierte.

Eine der wegweisenden Innovationen in dieser Version war die Integration von OpenGL, was zu einer erheblichen Verbesserung der Grafikleistung führte. Dieser Schritt legte den Grundstein für zukünftige Entwicklungen in der visuellen Darstellung von macOS.

Mac OS X 10.2 (Jaguar)
24.08.2002

macOS 10.2 baute auf den Erfolg von Puma auf und führte innovative Funktionen ein, die die Benutzererfahrung weiter verbesserten. Der Quartz Extreme Grafik-Engine verlieh der grafischen Darstellung eine neue Dimension, während der Safari-Browser erstmals sein Debüt feierte.

Neben der Verbesserung der allgemeinen Geschwindigkeit und Leistung brachte Jaguar auch Funktionen wie das Adressbuch und iChat auf den Mac. Diese Integration von Kommunikations- und Organisationswerkzeugen trug dazu bei, dass macOS nicht nur funktionaler, sondern auch benutzerfreundlicher wurde.

Mac OS X 10.3 (Panther)
24.10.2003

macOS 10.3 revolutionierte die Benutzeroberfläche mit der Einführung von Exposé und einem neuen Finder. Die Integration von iChat AV und Safari als Standardbrowser waren weitere Meilensteine dieser Version.

Eine bemerkenswerte Neuerung war die Integration von Zugangsdaten für soziale Netzwerke in die Systemeinstellungen. Dies erleichterte den Zugriff und die Interaktion mit sozialen Plattformen direkt über das Betriebssystem.

Mac OS X 10.4 (Tiger)
29.04.2005

Mit der Veröffentlichung von macOS 10.4 setzte Apple den Fokus auf Stabilität und führte gleichzeitig eine Reihe bahnbrechender Funktionen ein. Die Einführung von Widgets, Spotlight-Suche und dem Automator-Automatisierungstool hoben die Effizienz und den Funktionsumfang von macOS auf ein neues Niveau.

Neben der Unterstützung des neuen Spotlight-Suchsystems und Safari RSS ermöglichte Tiger auch die direkte Platzierung von Widgets auf dem Desktop mit Dashboard. Diese visuellen Elemente verliehen dem Betriebssystem nicht nur eine ansprechende Ästhetik, sondern verbesserten auch die Benutzerfreundlichkeit.

Mac OS X 10.5 (Leopard)
26.10.2007

macOS 10.5 war eine wegweisende Version von macOS mit zahlreichen neuen Funktionen. Time Machine, Spaces für virtuelle Desktops und Boot Camp für die Installation von Windows auf einem Mac waren Schlüsselfunktionen.

Die überarbeitete Benutzeroberfläche und die verbesserte Stabilität machten Leopard zu einer der stabilsten und funktionsreichsten Versionen von macOS. Diese Version trug dazu bei, dass macOS nicht nur für den Alltagsgebrauch, sondern auch für professionelle Anwendungen noch attraktiver wurde.

Mac OS X 10.6 (Snow Leopard)
28.08.2009

macOS 10.6 konzentrierte sich auf Leistungsverbesserungen und Stabilität. Die Einführung von Grand Central Dispatch und OpenCL zielt darauf ab, die Leistung auf Multi-Core-Systemen zu maximieren.

Diese Version markierte auch das Ende der Unterstützung für PowerPC-Software, was bedeutete, dass Entwickler ihre Anwendungen an die neue Architektur anpassen mussten. Trotzdem festigte Snow Leopard den Ruf von macOS als eine Plattform für zuverlässige Leistung und Effizienz.

OS X 10.7 (Lion)
20.07.2011

macOS 10.7 brachte viele iOS-Elemente auf den Mac und ebnete den Weg für eine nahtlose Integration beider Plattformen. Die Einführung von Launchpad und Mission Control verbesserte die Übersicht und das App-Management.

Zusätzlich zur Integration von iOS-Elementen führte Lion auch den Mac App Store ein und unterstützte Vollbild-Apps. Funktionen wie das Wiederherstellen von Versionen und die Einführung von AirDrop für die Dateifreigabe betonten die Benutzerfreundlichkeit dieser Version.

OS X 10.8 (Mountain Lion)
25.07.2012

Mountain Lion vertiefte die Integration von iOS und führte iMessage, Notification Center und Game Center auf den Mac ein. Power Nap aktualisierte den Mac auch im Ruhezustand, während die Integration von Facebook und Twitter in die Systemeinstellungen eine reibungslose soziale Interaktion ermöglichte.

Die kontinuierliche Annäherung an iOS festigte Apples Position als Anbieter einer kohärenten Erfahrung über all seine Plattformen hinweg.

OS X 10.9 (Mavericks)
22.10.2013

Mavericks konzentrierte sich auf die Verbesserung der Batterielaufzeit und Gesamtleistung, was insbesondere für MacBook-Benutzer von Vorteil war. Die Einführung von iBooks und Apple Maps auf dem Mac erweiterte die Multimedia-Funktionalität.

Zusätzliche Verbesserungen, darunter Finder-Tabs und Tags, trugen dazu bei, dass Mavericks nicht nur effizienter, sondern auch besser organisiert war. Diese Version betonte die kontinuierlichen Bemühungen von Apple, die Benutzererfahrung auf allen Ebenen zu optimieren.

OS X 10.10 (Yosemite)
16.10.2014

Yosemite brachte eine radikale Überarbeitung des Designs von macOS mit sich, um eine nahtlose Integration mit iOS zu ermöglichen. Die Benutzeroberfläche erhielt ein frisches, flaches Design mit Transparenzen und neuen Icons.

Diese visuellen Veränderungen verliehen macOS eine moderne Ästhetik, die konsistent über alle Apple-Geräte hinweg war. Yosemite markierte nicht nur einen ästhetischen Wandel, sondern legte auch den Grundstein für eine engere Integration von Funktionen und Daten zwischen Mac und iOS.

OS X 10.11 (El Capitan)
30.09.2015

El Capitan baute auf Yosemite auf und brachte signifikante Verbesserungen in Bezug auf Benutzerfreundlichkeit und Leistung. Die Einführung der Metal-Grafiktechnologie optimierte die Ressourcennutzung und trug zu einer insgesamt verbesserten grafischen Performance bei.

El Capitan demonstrierte Apples Engagement für kontinuierliche Verbesserungen und Optimierungen. Diese Version legte den Grundstein für zukünftige Innovationen und bekräftigte die Position von macOS als eine der fortschrittlichsten Betriebssystemplattformen.

macOS 10.12 (Sierra)
20.09.2016

Sierra integrierte Siri, den virtuellen Assistenten von Apple, direkt in macOS. Die iCloud-Synchronisation wurde optimiert, und Funktionen wie Auto Unlock ermöglichten es Benutzern, sich nahtlos zwischen ihren Apple-Geräten zu bewegen.

Eine weitere bemerkenswerte Einführung war das Project Catalyst, das es Entwicklern erleichterte, iPad-Apps auf den Mac zu bringen. Diese strategische Annäherung an iOS und iPadOS verstärkte die nahtlose Integration von Apple-Produkten weiter.

macOS 10.13 (High Sierra)
25.09.2017

High Sierra führte das Apple Dateisystem (APFS) ein, was zu einer verbesserten Leistung und Sicherheit beitrug. Grafik und Virtual Reality erhielten Upgrades, die die Nutzung von leistungsintensiven Anwendungen ermöglichten.

Die kontinuierlichen Bemühungen von Apple um Innovation und Verbesserung waren in High Sierra deutlich sichtbar. Die Einführung von APFS und die Optimierung der grafischen Leistung unterstreichen Apples Bestreben, macOS ständig auf dem neuesten Stand der Technik zu halten.

macOS 10.14 (Mojave)
24.09.2018

Mojave führte den Dark Mode, Create ML, und die Abkehr von OpenGL und OpenCL ein. Sicherheitsupdates ermöglichten detaillierte Freigaben und begrenzten Website-Fingerprinting. Neue Desktop-Features, überarbeiteter Mac App Store, Facetime-Gruppengespräche, Tethering mit iPhone-Kamera und die Home-App für HomeKit prägten Mojave. Es war die letzte Version, die 32-Bit-Anwendungen unterstützte, und wurde bis Anfang 2021 unterstützt.

macOS 10.15 (Catalina)
07.10.2019

Catalina führt als reines 64-Bit-System weg von 32-Bit-Anwendungen. Die System Integrity Protection ist verstärkt, Kernprogramme laufen auf einem Nur-Lese-Volume. Kein Platz mehr für 32-Bit-Programme, darunter die Carbon-Schnittstelle und QuickTime 7. Aperture wird nicht mehr unterstützt, ebenso entfällt iTunes. Neu sind die Umsetzung von iPadOS-Apps, Sidecar als zweiter Bildschirm, Sprachsteuerung und Bildschirmzeit-Anzeige. Ruby, Python und Perl sind nicht mehr vorinstalliert. Catalina steht für Sicherheit, Innovation und klare Ausrichtung.

macOS 11.0 (Big Sur)
12.11.2020

Big Sur brachte ein überarbeitetes macOS-Design mit neuen Systemtönen, vereinten Fensterleisten und einem Kontrollzentrum in der Menüleiste. Technologisch fortschrittlich unterstützte es den Apple M1-Prozessor und ermöglichte die nahtlose Ausführung von iOS-Apps auf Macs.

Das Systemvolumen erhielt kryptografische Signierung, Softwareaktualisierungen wurden im Hintergrund durchgeführt, und Spotlight wurde beschleunigt. Safari erhielt ein neues Erscheinungsbild und Funktionen wie personalisierbare Startseiten und Tab-Übersetzungen.

Big Sur brachte Veränderungen in Standard-Apps mit sich, darunter die Übertragung der Nachrichten-App von iOS zu macOS. Trotz dieser Innovationen stieß das Betriebssystem auf Datenschutzbedenken aufgrund der Kommunikation mit Apple-Servern bei jeder Programmöffnung.

macOS 12.0 (Monterey)
25.10.2021

Monterey hob die Integration mit iOS auf eine neue Ebene, ermöglichte verbesserte Cross-Device-Funktionalität und brachte Funktionen wie Live Text und AirPlay to Mac. Das Betriebssystem betonte nahtlose Konnektivität zwischen verschiedenen Apple-Geräten.

Monterey präsentiert diverse Neuerungen, darunter die Einführung der Kurzbefehle-App als Ersatz für Automator. TestFlight, zuvor nur auf iOS verfügbar, ermöglicht nun auch das Testen von Beta-Apps auf dem Mac.

Apple setzt den angekündigten Wegfall von Unix-Skriptsprachen um, wobei in der ersten Beta-Version die Skriptsprache PHP für dynamische Webseiten fehlt. Safari passt seine Fenstertitelleiste an die aufgerufene Webseite an und bietet eine Tab-Gruppierungsfunktion. Universelle Steuerung erlaubt die Nutzung von Peripheriegeräten auf bis zu drei Macs und iPads.

macOS 13.0 (Ventura)
24.10.2022

Ventura revolutioniert die Benutzererfahrung mit innovativen Funktionen. Key-Features umfassen den Stage Manager, der Fenster effizient organisiert, Spotlight mit Datei-Vorschau, rückgängig machbare E-Mails in Apple Mail und geteilte Safari-Tabs für Zusammenarbeit. Metal-Verbesserungen erlauben iPhones als Webcams. Die "Einstellungen"-App ist an iOS angeglichen. Neu sind die Uhr-App mit Multi-Zeitzonen-Ansicht und Timer/Wecker sowie die kollaborative Freeform-App. Ventura bietet ein umfassendes Upgrade für eine nahtlose, verbesserte Nutzung.

macOS 14.0 (Sonoma)
26.9.2023

Sonoma bringt Widgets mit flexibler Platzierung auf dem Schreibtisch, die bei geöffneten Fenstern dezent verblassen. Ein spezieller Spielmodus optimiert Spiele auf der Hardware, begleitet von einem Game Porting Toolkit für einfachere Windows-Spielportierungen. Neue Desktop-Hintergrundbilder und Videohintergründe für FaceTime betonen Sonomas visuelle Innovationen.
